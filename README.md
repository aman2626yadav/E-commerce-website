
# E-Commerce Website | Dockerized Web Application

#Tech Stack: HTML5, CSS3, JavaScript, Docker, Nginx, Ubuntu

Developed and containerized a responsive E-Commerce web application featuring dynamic product listing and client-side cart functionality. Designed a clean and user-friendly interface using HTML, CSS, and JavaScript.

Implemented Docker containerization using an Ubuntu-based image with Nginx to deploy the application in an isolated and portable environment. Configured port mapping and optimized the Dockerfile for lightweight deployment.

Deployed the containerized application on AWS EC2 and configured security group rules to enable external access via HTTP (Port 80).

🔹 Key Contributions:

Built a responsive front-end shopping interface

Containerized the application using Docker

Configured Nginx server inside container

Deployed on AWS EC2 instance

Managed port mapping and container lifecycle
